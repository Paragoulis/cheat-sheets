## Getting started
```
$ chsh $USER
...
New shell [/bin/bash]: /bin/zsh
Shell changed.
```
```
$ echo $SHELL
/bin/zsh
```
```
$ mv ~/.zshrc ~/.zshrc.bak
$ clear
$ touch ~/.zshrc
```
